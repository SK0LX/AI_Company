# Сервисы для ИИ-звонков: как настроить, чтобы тебе на телефон позвонил ИИ

> Обзор актуален на 2026 год. Цены и доступность перепроверены веб-поиском.
> Для РФ-пользователя решающий фильтр — **не цена, а две вещи: чем платить и дойдёт ли звонок на +7.**

---

## Коротко: 3 способа «чтобы тебе позвонил ИИ»

1. **Готовые платформы (turnkey)** — регистрируешься, дёргаешь один API, телефон звонит. Минуты на запуск.
2. **Собрать из кирпичей** — телефония (Twilio/Telnyx) + «мозг+голос» (OpenAI Realtime / ElevenLabs / Deepgram) + оркестратор (LiveKit/Pipecat). Для разработчика.
3. **Российские сервисы** — рубли, родной дозвон на +7, русская речь из коробки.

## 2. Собрать из кирпичей (для разработчика)

Логика: **телефония** (дозвон) + **мозг+голос** (разговор) + при желании **оркестратор**.

### Телефония (звонит реально)
- **[Twilio](https://www.twilio.com/en-us/guidelines/ru/voice)** — дефолт. +7 официально Yes/Yes в обе стороны, но **выключен по умолчанию** (антифрод), включается вручную. Звонок на моб. +7 ≈ $0.34/мин. `ConversationRelay` +$0.07/мин даёт готовый мост к твоему LLM.
- **[Telnyx](https://telnyx.com/products/voice-api)** — свой телеком-оператор, в 2025 добавил 2-way РФ; Voice AI-агент от ~$0.05/мин.

### Мозг + голос
- **[OpenAI Realtime](https://platform.openai.com/docs/guides/realtime)** (`gpt-realtime`) — speech-to-speech, отлично говорит по-русски. ~$0.06/мин вход + ~$0.24/мин выход. Сам **не звонит** — нужен Twilio-мост ([готовый туториал](https://www.twilio.com/en-us/blog/outbound-calls-python-openai-realtime-api-voice)).
- **[Deepgram Voice Agent](https://deepgram.com/product/voice-agent-api)** — STT+LLM+TTS в одном WS, ~$0.075/мин. **Важно: его TTS (Aura) не умеет русский** — подставляй ElevenLabs для голоса.

### Оркестраторы (open-source, можно self-host бесплатно)
- **[LiveKit Agents](https://docs.livekit.io/telephony/)** — WebRTC-первый фреймворк со встроенным SIP-мостом.
- **[Pipecat](https://www.pipecat.ai/)** — Python-фреймворк, самая широкая библиотека интеграций.

Оба сами не продают +7-номера — подключаешь любой trunk. Self-host = платишь только за trunk + модели (или вообще ничего, если модели локальные).

### Компоненты речи (оплата рублями, но сами НЕ звонят)
- **[Yandex SpeechKit](https://yandex.cloud/ru/docs/speechkit/pricing)** — топовый русский ASR/TTS, есть call-center модель.
- **[Sber SaluteSpeech](https://developers.sber.ru/docs/ru/salutespeech/tariff)** — есть free-тариф для физлиц (100 мин ASR + 200k символов TTS/мес).
- **[T-Bank VoiceKit](https://www.tbank.ru/software/voicekit/)** — посекундная оплата, отличный русский.

К ним нужна телефония, чтобы реально набрать номер.

### SIP-trunk'и для дозвона на +7 (рубли)
- **[Novofon](https://novofon.com/prices/)** — от ~0.44 ₽/мин, оплата рублями/ЮMoney, **лучший вариант** (нюанс: для физлица регистрация требует УКЭП).
- **[SIPNET](https://www.sipnet.ru/tarify-ip-telefonii/russia)** — ~1.6 ₽/мин на моб.
- **MCN Telecom** — бизнес-ориентированный, нужен статический IP.

## ⚠️ Главный практический момент для РФ

Перепроверка свела всё к двум барьерам на западных сервисах (Bland, Vapi, Retell, Twilio, OpenAI, ElevenLabs):

1. **Оплата.** Все биллят USD/EUR-картой. Российские карты не работают за рубежом с 2022, а в 2025 их трансграничный приём добивают окончательно. → Нужна **зарубежная карта** (Казахстан/Армения/Грузия/Турция/ЕС) или посредник.
2. **Дозвон на +7.** Западные операторы держат РФ выключенной по умолчанию как высокорисковое направление (IRSF-фрод) — включается вручную и стоит дорого.

→ Для РФ-разработчика **родной SIP-trunk бьёт Twilio**, а готовые РФ-роботы бьют Bland/Vapi.

### Про закон
С 01.09.2025 (41-ФЗ + ПП №1300) обязательна **маркировка звонков** и запрет массового обзвона без согласия (штрафы до 1 млн ₽ для юрлиц). Но если ИИ звонит **на твой собственный номер** — это не массовый и не рекламный обзвон, всё легально. Авто-дозвон по чужим без согласия — запрещён.

## Что выбрать под «чтобы мне позвонил ИИ»

| Ситуация | Решение |
|---|---|
| Просто быстро попробовать, **есть зарубежная карта** | **Bland AI** — буквально один curl (см. ниже) |
| **Без зарубежной карты**, нужен дозвон на +7 | Готовый РФ-сервис (**Neuro.net «Небо»** или **Звонобот** — самый быстрый старт) |
| Полностью без зарубежной оплаты, своими руками | **Novofon SIP-trunk + Pipecat/LiveKit** + self-host мозг (Whisper + LLM + русский TTS) |
| Нужен максимально естественный **русский голос** | **ElevenLabs Agents** + русский SIP-trunk для +7 |

### Быстрый пример — Bland AI (нужна зарубежная карта)

```bash
curl -X POST https://api.bland.ai/v1/calls \
  -H "authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"phone_number": "+7XXXXXXXXXX",
       "task": "Ты — мой ассистент. Позвони и напомни про встречу в 15:00, спроси, перенести ли."}'
```
